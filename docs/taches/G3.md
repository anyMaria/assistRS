## Objectif de ce bloc

Recherche visuelle par thème sur Pinterest/Instagram/LinkedIn/Facebook/Meta Ad Library,
avec cache 14 jours et budget mensuel maîtrisé (~5 €/mois), et épinglage en moodboards.

## Ce qu'il faut construire

- Schéma : `inspiration_searches`, `inspiration_items`, `moodboards`, `api_usage`
  (colonnes exactes dans `CONCEPTION.md` §10) → `npm run db:push`.
- `src/lib/apify.ts` : helper(s) + constantes de coûts par actor (`GUIDELINE.md` §6).
  **Implémente le pattern asynchrone** (démarrage du run, polling toutes les 5 s sur une
  route `GET /api/inspiration/statut?runId=...`, page en état « Recherche en cours… »
  avec skeleton) — ne pas utiliser l'endpoint synchrone qui dépasserait le timeout Vercel.
- Page `/s-inspirer` : thème + marque optionnelle + cases sources (Pinterest pré-cochée,
  coût estimé affiché à côté de chaque source) + filtres (période, tri). Résultats en
  grille masonry : image, auteur, signaux de performance par source (voir tableau
  `CONCEPTION.md` §5.2), lien original, bouton « Épingler ».
- Barre de budget en haut : `monthlySpendCents("apify")` / 500 ¢ ; blocage doux + message
  de confirmation au-delà de 450 ¢.
- Cache 14 jours : avant tout appel, chercher une recherche identique (thème normalisé +
  source + filtres) < 14 jours → resservir avec mention « résultats du JJ/MM (cache
  gratuit) » + bouton « Relancer ».
- Épingler : téléchargement serveur de l'image → Vercel Blob privé (`blobThumbUrl`) →
  association à un moodboard (existant ou créé à la volée, par marque ou libre/thème).
  Page `/s-inspirer/moodboards` en grilles.
- LinkedIn : uniquement `harvestapi/linkedin-post-search` (recherche de posts) — **ne
  jamais** implémenter de scraping de profils.

## Critères d'acceptation

- [ ] Sans `APIFY_TOKEN` : message propre, page utilisable.
- [ ] Épingler copie l'image en Blob et l'affiche dans le moodboard.
- [ ] Le blocage doux à 4,50 $ se déclenche (testable en insérant des lignes `api_usage`).
- [ ] `npx tsc --noEmit` et `npm run build` passent, G1 (et G2 si fait) intacts.
