## Ce qu'il faut construire

- Recherche ⌘K : composant client global (dialog), route `GET /api/recherche?q=` → LIKE
  sur marques, idées, publications, moodboards, modèles ; résultats groupés par type,
  navigation clavier.
- Bouton « + » global (coin bas droit desktop / dans la barre mobile) : menu 4 actions
  (idée, publication, relevé, temps).
- Page 404 personnalisée, états vides soignés partout, revue accessibilité (labels,
  `alt`, `role="dialog"`, focus, `prefers-reduced-motion`).
- `README.md` : installation locale, variables d'environnement, déploiement pas à pas
  Vercel + Turso + Blob + Resend + Apify.
- PWA : `manifest.json` + icônes + meta iOS (`apple-mobile-web-app-capable`) —
  installable sur iPhone. Web push reporté, non bloquant.
- Vérification finale complète (parcours connexion → marque → idée → publication →
  relevé → bilan), idéalement outillée (Playwright si disponible dans l'environnement).
