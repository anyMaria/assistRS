# Tâche G8 — Refonte visuelle : nouveau thème, icônes, shell de navigation

## Avant de commencer (obligatoire)

- Lis `docs/GUIDELINE.md` §11 (Pièges connus) et §12 (Définition de « terminé »).
- Clone la maquette de référence : `https://github.com/anyMaria/CrErMaquetteSiteWeb`
  (export Figma Make, Vite + React + shadcn). **C'est la référence visuelle unique**
  de ce bloc. Regarde surtout : `src/styles/theme.css` (tokens), `src/app/components/layout/*`
  (Sidebar, MobileNav, AppLayout), `src/app/components/ui-atelier/Primitives.tsx`.
- La maquette est un mock front-only : on **copie son langage visuel**, pas son code
  tel quel (elle est en react-router + mocks ; l'app reste Next.js App Router + Drizzle).
- Modèle recommandé : **Sonnet**.

## Objectif de ce bloc

Remplacer l'ancien style « atelier éditorial » (papier crème, bordures 2px encre,
angles droits, ombres dures, icônes unicode ◆✎☁) par le style de la maquette :
moderne, épuré, arrondi, avec de vraies icônes. **Aucun changement fonctionnel** dans
ce bloc — mêmes pages, mêmes routes, mêmes actions.

## 1. Tokens de thème (globals.css / theme Tailwind v4)

Reprendre exactement les tokens de `src/styles/theme.css` de la maquette :

- Clair : `paper #f5f6f8`, `paper-2 #eceef3`, `surface #ffffff` (cartes), `ink #0e1116`,
  `line #e6e8ef` (bordures), `accent #f5352b` (hover `#d62419`), `accent-2 #ff6a3d`,
  `accent-soft #ffe9e7`, dégradé accent `linear-gradient(120deg,#f5352b,#ff6a3d)`,
  `danger #e5484d`, `warn #f5a524`, `ok #17b26a`, plateformes inchangées.
- Sombre (classe `.dark` sur `<html>`, à persister en `localStorage`) : `paper #0b0d12`,
  `paper-2 #171a22`, `surface #14171e`, `ink #eef1f6`, `line #262b36`, `accent #ff4a3d`.
- Typo : **Space Grotesk** (titres, `--font-display`) + **Inter** (corps) — remplace
  Fraunces/Work Sans. Plus de titres en italique.
- `--radius: 0.875rem` (arrondis généreux partout). **Supprimer** toutes les ombres
  dures `shadow-[3px_3px_0_#1C1917]` et les bordures `border-2 border-ink` → bordure
  `1px solid var(--line)` + fond `surface`.

Adapter les classes utilitaires existantes (`.btn`, `.btn-accent`, `.tag`, `.card`,
`.field`, `.field-label`) aux nouveaux tokens plutôt que de toucher chaque page :
une grande partie de la refonte passe par là.

## 2. Icônes et boutons d'action

- Installer `lucide-react`. Remplacer **tous** les glyphes unicode (◆ ✎ ☁ ▦ ✺ ◔ ▁▃▅ ◎ ✉ ☾ ▣ ⚙ ⟳ ＋)
  par les icônes lucide équivalentes (voir `nav-items.ts` de la maquette :
  Home, Palette, PenLine, Sparkles, CalendarDays, BarChart3, Mail, Settings…).
- Règle maquette : **un seul bouton accent (rouge) par écran** — l'action principale.
  Les autres boutons : variante neutre (bordure `line`, fond `surface`, hover `paper-2`).
- **Supprimer les liens texte soulignés « Supprimer »** partout : les remplacer par un
  bouton-icône `Trash2` bien visible (taille min 36×36 px) ouvrant une confirmation
  (`AlertDialog` ou `confirm()` propre). Idem « Dupliquer » → icône `Copy` avec tooltip
  (`title=`). Les mini-liens « ✎ modifier » deviennent des boutons-icônes `Pencil`.
- Les `<details>/<summary>` utilisés comme accordéons de formulaires (« + Nouvelle idée »,
  « + Nouvelle publication », « ⟳ Récurrences ») deviennent des **boutons** dans l'en-tête
  de page qui ouvrent un **Dialog/panneau** (pattern `SectionHeader` avec `action` à
  droite, comme `Planifier.tsx` de la maquette).

## 3. Shell de navigation

- Reprendre le pattern `AppLayout` de la maquette : **sidebar fixe à gauche** (desktop)
  avec logo, items icône+libellé, item actif fond `accent-soft` texte `accent` ;
  **topbar** fine avec recherche ⌘K, bouton « + » de capture rapide et bascule
  clair/sombre (`ThemeToggle`) ; **barre de nav mobile en bas** (5 items max, cf.
  `mobile: true` dans `nav-items.ts`).
- La liste des entrées de nav ne change **pas encore** dans ce bloc (elle sera
  restructurée en G9) — seule l'apparence change.

## 4. En-têtes de section

- Créer un composant `SectionHeader` (titre display 2xl-4xl, sous-titre `ink/55`,
  zone `action` à droite pour le bouton principal) et l'utiliser sur les 12 pages.
  Le titre de l'Accueil devient personnel : « Bonjour Ana » avec dégradé accent
  (cf. `Accueil.tsx` maquette, prop `gradient`).
