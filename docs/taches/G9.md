## 2. Page Bilan unifiée (rapports)

Une seule page `/bilan` avec des **onglets** (composant Tabs, pattern maquette) :

- **Semaine** : le digest hebdo actuel (deadlines, à relancer, à relever, highlights).
- **Mois** : le contenu actuel d'Analyse *utile* — objectifs (création/suivi), temps
  passé, rapports PDF mensuels par marque. Le reste d'Analyse (analyse d'horaires
  `computeRealSlots`) part en G10 dans le planning ; ne pas le réimplémenter ici.
- **Rituel** : le wizard mensuel actuel, inchangé, simplement déplacé.
  La bannière d'invitation au rituel (dashboard, 7 premiers jours du mois) pointe
  désormais vers `/bilan?onglet=rituel`.

Les routes `/analyse` et `/rituel` redirigent (308) vers les onglets correspondants.
Les e-mails (bilan hebdo, invitation rituel) mettent à jour leurs liens.

« configurée / manquante » par variable d'env, sans jamais afficher les valeurs).
